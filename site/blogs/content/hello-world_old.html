Welcome! This is a small, BBC‑inspired personal site built to be fast, responsive, and hostable on GitHub Pages. The goal is to keep things simple and readable while hinting at the familiar layout patterns of a large news site.

From here, we’ll iterate: more posts, theming, and maybe a light CMS in the future. For now, it’s all static HTML, CSS, and a tiny bit of JavaScript for the navigation menu.

What’s included now

  • Responsive layout with a bold, dark header.
  • Three sections: Blogs, Images & Videos, Demos.
  • GitHub Actions workflow to deploy to GitHub Pages.
  • JSON-driven Images & Videos, and now Blogs.

Thanks for visiting — enjoy exploring the sections and samples!
